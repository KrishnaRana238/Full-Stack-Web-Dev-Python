Background Image
This example demonstrates how to set a background image that covers the entire viewport.
CSS Properties Used
  • background-image: Sets the image to be used as the background.
  • background-size: cover: Ensures the image covers the entire area.
  • background-position: center: Centers the image in the viewport.
  • height: 100vh: Sets the height to 100% of the viewport height.
  • margin: 0: Removes default margin around the body.
